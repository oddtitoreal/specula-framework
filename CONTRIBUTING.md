# Contributing (Private)

This repository is delivered for private client use.

## Change workflow
1. Open an issue describing the change.
2. Create a branch: `docs/<short-name>` or `specs/<short-name>`.
3. Update the relevant document(s) **and** `CHANGELOG.md`.
4. Submit a pull request.

## Review rules
- Normative docs require 2 reviewers.
- Non-normative docs require 1 reviewer.
